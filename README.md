# Estudos de Modelagem Relacional em MySQL

Este repositório contém meus estudos práticos de SQL e modelagem de dados utilizando MySQL Workbench.

## 🛠️ Tecnologias e Conceitos Aplicados
* **SGBD:** MySQL
* **DDL (Data Definition Language):** Criação de tabelas (`CREATE TABLE`), definição de tipos de dados (`VARCHAR`, `INT`), e uso de *Constraints* (`NOT NULL`, `UNIQUE`).
* **DML (Data Manipulation Language):** Inserção de dados (`INSERT INTO`) e consultas básicas (`SELECT`).
* **Gerenciamento:** Criação e deleção de estruturas com validação (`IF NOT EXISTS`, `DROP TABLE`).

## 🎯 Objetivo
Projeto desenvolvido para consolidar a base de estruturação de bancos de dados relacionais, um passo fundamental na minha transição para a área de Engenharia de Dados e Software.

# Relational Data Modeling Studies in MySQL

This repository contains my practical studies of SQL and data modeling using MySQL Workbench.

## 🛠️ Technologies and Applied Concepts
* **RDBMS:** MySQL
* **DDL (Data Definition Language):** Table creation (`CREATE TABLE`), data type definition (`VARCHAR`, `INT`), and usage of Constraints (`NOT NULL`, `UNIQUE`).
* **DML (Data Manipulation Language):** Data insertion (`INSERT INTO`) and basic queries (`SELECT`).
* **Management:** Creation and deletion of structures with validation (`IF NOT EXISTS`, `DROP TABLE`).

## 🎯 Objective
Project developed to consolidate the foundational structure of relational databases, a critical step in my transition to Data and Software Engineering.
